Library & File System

When you add an item, a folder is auto-created in your library (e.g. Movie (2026)/). Drop your real file there with a .real extension (e.g. Movie.mkv.real). Jellyfin ignores it. At the scheduled time, the plugin removes the .real extension — your movie appears instantly.
Optional: Create a dummy MKV for Jellyfin to pre-fetch metadata (poster, cast, etc.) before the real file arrives.
This folder must already be set up as a Movie or TV library in Jellyfin.
